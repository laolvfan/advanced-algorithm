# 《Succincter》论文综述

本仓库用于完成 2026 年春季学期《高级算法》课程中关于 Mihai Pătrașcu
论文 *Succincter* 的综述大作业。

## 仓库内容

- `Succincter.pdf`：本次综述所选论文的原文
- `notes/`：阅读笔记与资料来源记录
- `draft/`：逐步撰写和修改的综述草稿
- `ai-usage/`：可追溯的 AI 辅助使用记录

## 协作约定

所有工作都应分阶段提交，使仓库历史能够呈现从收集资料、阅读论文、形成观点，
到撰写和反复修改草稿的完整过程。每位成员应使用自己的身份提交贡献。AI 只能
辅助理解或查询信息，不得代写最终提交的综述；每次使用 AI 都必须留下记录，并
对所得信息进行独立核验。
